Пакет утилит для обработки и анализа динамической КТ

image - утилиты для работы с файлами
----- dcm_parser - парсинг и копирование dcm файлов с формированием субдерикторий в соответствии с именем пациента, датой исследования, сериями
----- nii_separator - разделение 4d nii файла на 3d

filters - обработка изображений

----- bilateral - билатеральнный фильтр с 3d кернелем реализован на Cython

roi - автоматический подсчёт параметров выделенных зон интереса
----- distance - получение карты расстояний для ROI
----- density - генерация csv файла КТ чисел из списка ROI и списка соответствующих им 3d nii
